<h1>TANK BLITZ</h1>

It is grid based top-down tank battle survival game.
The rules are simple:
1. Defeat other enemies to survive.
2. Shoot other enemies to increase the score.

<h3>Controls:</h3>
<p>1. Arrow Keys to Move</p>
<p>2. Space Bar to Shoot</p>
